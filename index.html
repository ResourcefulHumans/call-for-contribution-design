    Propose a contribution

    At Resourceful Humans we work based on contributions. Everybody needs to be clear on what their contribution towards our overall goal is: turn organizations into a network of entrepreneurs.



    You set the scope

    That's why we depend on you to set the scope for this project. We expect you to outline what you want to achieve, in what timeframe and what that will cost us.


    We plan to focus our own resources on this project in the time between Sep 18th and 30th 2016.

    We are looking for long-term contributers

    Great design and usability is an important building block for successful digital products. We see this project as an opportunity to find new Resourceful Humans which we will likely invite to a permanent position if they are a match personally and professionally.

    What we expect

    The result of this project should contain the answers to these questions:

    - How does an RH website look like

    - How does an RH web application look like

    - How does an RH mobile application look like

    - How does an RH transactional email look like
